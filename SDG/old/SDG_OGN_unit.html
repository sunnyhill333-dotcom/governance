Component (Organism이 소비)
Page (Organism을 소비)
SDG_OGN_RULE_1
다중 요소로 구성된 Component 단위가 Organism이 됩니다.
둘 이상의 Component가 하나의 서비스 기능을 위해 결합될 때 Organism으로 분류합니다.
SDG_OGN_RULE_2
Organism은 전체 계층에서 최상위 Component 레벨을 담당합니다.
Foundation → Component → Organism → Page 순서의 계층에서, Organism은 Component를 소비하고 Page에 소비됩니다.
SDG_OGN_RULE_3
Organism은 단독으로 사용되어야 합니다.
Organism 내 Component가 하위로 구성됩니다. Local_ 연결이 필요한 경우 별도 등록합니다.
SDG_OGN_RULE_4 — 주의
단일 요소만으로 구성된 경우 Organism으로 분류하지 않습니다.
⚠️ Component 하나만 감싼다고 Organism이 되지 않습니다. 반드시 2개 이상의 Component가 조합되어야 합니다.
